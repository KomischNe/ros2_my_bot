## System Requirements

- **Operating System:** Ubuntu 24.04.3 LTS  
- **ROS2 Distribution:** Jazzy  

### ROS2 Installation

Run the following command in your terminal to install ROS2 Jazzy:

```bash
wget http://fishros.com/install -O fishros && . fishros
```
